Docker

Tome provides Docker images for easily creating and running Tome sites. The Docker images are hosted on Docker Hub at mortenson/tome and developed on Github at drupal-tome/tome-docker.

To create a new Tome project using Docker, run:

mkdir my-project
cd my-project
docker run --rm -it -v "$(pwd)":/var/www/tome mortenson/tome init

To start a local server, run:

docker run --rm -it -p 8888:8888 -v "$(pwd)":/var/www/tome mortenson/tome

To generate static HTML, run:

docker run --rm -it -v "$(pwd)":/var/www/tome mortenson/tome drush tome:static
